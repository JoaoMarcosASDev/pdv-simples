# PDV Simples

Minha mãe possui uma loja que não possui software para gerênciar estoque, calcular o faturamentos, analisar os dados, etc...

Tendo em visto esse problema, busco desenvolver um programa que cumpre esses papeis para facilitar e melhorar o negócio, e para praticar e aprender sobre o desenvolvimento de software

### O que é necessário
* Armazenamento de Banco de Dados local com possibilidade de migrar para a nuvem:
    Armazenamento:
    - Local: SQLite 3
    - Em Nuvem: PostgreSQL
* Interface visual web (Front-end):
    Tecnologias:
    - HTML
    - CSS (Possivelmente substituir por algum framework de estilização)
    - JavaScript
* Intermediário da interface visual e banco de dados (Back-end):
    Tecnologia :
    - NodeJs

> [!NOTE]
> Este projeto visa desenvolver uma base sólida, então serão evitado o uso de Biblioteca externas, porém posso utilizar se necessário.
